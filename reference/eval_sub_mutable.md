# Homomorphic subtract, mutable variant

Homomorphic subtract, mutable variant

## Usage

``` r
eval_sub_mutable(x, ...)
```

## Arguments

- x:

  A `Ciphertext` (may be modified internally).

- ...:

  Method-specific arguments: `y` — a `Ciphertext` (may be modified
  internally).

## Value

A new `Ciphertext` holding `x - y`.
