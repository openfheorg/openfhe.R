# Serialize an OpenFHE object to file

Serialize an OpenFHE object to file

## Usage

``` r
fhe_serialize(x, ...)
```

## Arguments

- x:

  An OpenFHE object (CryptoContext, PublicKey, PrivateKey, Ciphertext)

- ...:

  Method-specific arguments (filename, format)

## Value

TRUE on success (invisibly)
