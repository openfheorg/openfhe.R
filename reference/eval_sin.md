# Evaluate sine on a ciphertext (Chebyshev approximation)

Evaluate sine on a ciphertext (Chebyshev approximation)

## Usage

``` r
eval_sin(ct, a, b, degree)
```

## Arguments

- ct:

  A Ciphertext

- a:

  Lower bound of the approximation interval

- b:

  Upper bound of the approximation interval

- degree:

  Chebyshev polynomial degree

## Value

A Ciphertext
